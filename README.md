# Open Clyo v0.1.0
## What is Clyo
Clyo is a personal advisor and assistant that runs on Notion AI. It lives inside your workspace and draws on the context you build with it over time, so its help is grounded in your actual situation, not generic advice. It's built to think with you, not just answer.
## What makes Clyo different
- **Grounded in your workspace:** loads your relevant context before it answers.
- **Pushes back:** names the strongest counter-argument instead of validating by default.
- **Memory that compounds:** a capture system (context database, profile, task journal) so you stop re-explaining yourself.
- **Built-in reliability guardrails:** drift logging, write-target safety, and co-design before it touches high-stakes docs.
- **Connects the dots:** links context across your projects and surfaces the blind spots you'd miss on your own.
**Open Clyo** is the free, open-source release you can install and adapt to fit how you think and work.
## What is in the package
- `README.md`: this overview.
- `clyo-instructions.md`: the instruction set that defines how Clyo thinks, responds, and operates.
- `deploy.md`: the one-prompt deploy text for guided setup.
- `setup-checklist.md`: the manual, step-by-step setup checklist.
- `how-to-use-clyo.md`: the bootstrapping doc that orients you once Clyo is installed.
- `LICENSE`: the MIT license.
## How to deploy
For the fastest path, use `deploy.md`: open a new Notion AI chat in your workspace and paste the full contents of `deploy.md` to run the guided agentic setup, which creates the surfaces and walks you through the manual steps. For manual step-by-step configuration, follow `setup-checklist.md`.
